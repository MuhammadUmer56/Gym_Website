# Gym Website

## Overview
This is a simple multi-page Gym Website built using **Flask**, **HTML**, and **CSS**.  
It includes pages for home, membership, and about us, along with a basic login/registration simulation (no database).

## Features
- Multi-page website (Home, Membership, About)
- Responsive design using CSS
- Flask backend for routing pages
- Simulated login and registration forms
